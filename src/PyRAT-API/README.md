### PyRAT API client

# Purpose: 
This is a Python client to interact with PyRat which is the database used to manage our mice at UCL. The script is for accessing mouse information and posting comments related to water delivery, experiments, surgeries and more.

# Requirements:
Modules to install (pip install):
- requests
- time
- json
- sys

API credentials: 
You'll need to contact Phill Newlands (p.newlands@ucl.ac.uk) and cc in Pip to be registered as part of the lab. Phill will provide the API client token and you can request a user token on PyRAT, please see the PyRAT documentation on the Wiki for more information if you get stuck.
